## install zenn-cli

- [Zenn 公式：Zenn CLI をインストールする](https://zenn.dev/zenn/articles/install-zenn-cli)

```sh
# check node version
$ node -v  # required: v13 or v14
v14.14.0

$ npm init --yes
$ npm install zenn-cli
$ npx zenn init  # setup directory for Zenn
```

## preview
